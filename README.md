# Bank Management System

A simple C++ bank management system using OOP, file handling, and CMake.

## Features
- Add Savings/Current Accounts
- Deposit/Withdraw Cash
- List All Accounts
- Persistent Storage

## Build & Run

```sh
mkdir build
cd build
cmake ..
make
./BankManagementSystem
```
